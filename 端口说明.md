# 本地运行与端口

## 换端口运行

项目已配置本地静态服务，默认端口 **5174**（避免和常见 5173/3000 冲突）。

- **默认端口 5174**  
  ```bash
  npm install
  npm run serve
  ```
  浏览器打开：<http://localhost:5174>

- **改用 3001 端口**  
  ```bash
  npm run serve:3001
  ```

- **改用 8080 端口**  
  ```bash
  npm run serve:8080
  ```

## 不用 npm 时

若未安装 Node，可用 Python 指定端口：

```bash
# Python 3
python3 -m http.server 5174
# 或 3001、8080 等
```

然后用浏览器打开：`http://localhost:5174`（或你选的端口）。

## 一直 loading 的说明

已做两处修改，减少卡在启动页的情况：

1. 启动页约 0.6 秒后就会进入，最多等 3 秒会强制进入。
2. 若脚本报错，会直接进入「首次设置」页，而不会一直停在 loading。

请用 **本地服务器** 打开（如 `npm run serve` 或 `python3 -m http.server`），不要直接双击 `index.html` 用 `file://` 打开，否则部分功能可能异常。
