## 🎯 必须实现的功能（基于雅威tech历史版本）

### 核心页面（7个）
1. **首页** 💕
   - 心情打卡（7种表情：😊😢😡🥰😴😐🤔）
   - 文字日记输入
   - 恋爱天数显示（自动计算）
   - 统计面板（日记数、心情分布）

2. **聊天** 💬
   - 与数字人Clark对话
   - 调用本地API：http://127.0.0.1:8081/v1/chat/completions
   - 模型：qwen/qwen3.5-9b
   - 语音合成：http://127.0.0.1:9880/tts
   - 语音文件：/tmp/openclaw_workspace/slx/clark_voice_8s_new.wav

3. **游戏** 🎮
   - 每日问答（随机问题库，至少20个问题）
   - 默契度测试（随机百分比+评语）
   - 恋爱指数进度条

4. **相册** 📸
   - 照片上传（URL或base64）
   - 网格展示
   - 点击放大查看

5. **账单** 💰
   - 共同记账（描述+金额）
   - 小金库功能（存取记录）
   - 收支统计图表

6. **愿望** ⭐
   - 愿望清单添加
   - 完成状态勾选
   - 进度百分比

7. **设置** ⚙️
   - 情侣名字设置
   - 头像更换（支持上传）
   - 开始日期设置（计算恋爱天数）
   - 主题切换（至少3种粉色系主题）
   - 背景音乐开关

### 新增创新功能（参考市场调研）

9. **打卡挑战** 🏆

   - 进度追踪

10. **年度回顾** 📊
    - 年度数据统计

    - 珍贵回忆自动整理

## 🎨 设计要求

### UI风格
- 粉色/紫色浪漫渐变主题
- 毛玻璃效果 (backdrop-filter: blur)
- 圆角设计 (border-radius: 16-28px)
- 柔和阴影
- 流畅动画（fadeIn、slideUp、heartbeat、pulse）

### 动画效果
- 启动页面动画（logo渐入+加载条）
- 心跳动画（💕 1.5s infinite）
- 气泡上升动画（背景装饰）
- 页面切换动画（fadeIn 0.4s）
- 按钮悬停效果（scale + shadow）
- 卡片悬停效果（translateY -5px）

### 响应式
- 手机优先（max-width: 520px）
- 桌面端侧边栏适配（min-width: 768px显示侧边栏）
- 底部导航栏（手机）/ 左侧导航（桌面）

## 💾 技术要求

### 数据存储
- LocalStorage 持久化
- 存储键名：
  - `love_diaries` - 日记列表
  - `love_moods` - 心情记录
  - `love_photos` - 相册
  - `love_bills` - 账单
  - `love_wishes` - 愿望
  - `love_settings` - 设置
  - `love_start_date` - 开始日期
  - `love_names` - 情侣名字

### PWA支持
- manifest.json
- Service Worker（离线缓存）
- 可安装到手机桌面

### 性能优化
- 图片懒加载
- 动画使用 CSS transform（GPU加速）
- 避免强制同步布局
